# Visual Studio Code 入門

## 環境設定について
- 環境設定画面の呼び出し  
```Ctrl+,```

- プログラミング用フォント**ricty-diminished** のインストール 
```command
sudo apt install fonts-ricty-diminished
```
- フォント **ricty-diminished** のVSCodeへの設定  
  **Edithor Fontfamily** に次を指定  
  ```Ricty Diminished```  

  再起動する

- タブスペースの数  
```Tab Size```

- スペースの表示
```Render Whitespace```

- 画面端で折り返し設定  
```Word Wrap```


## emmit について
